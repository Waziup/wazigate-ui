    LoRa sensors

    Those tutorials will help you build a low cost and low power sensor platform:

    • A simple LoRa IoT sensing device shows how to build a simple LoRa IoT sensing device with the simple temperature example and an Arduino Pro Mini, running on 4-AA battery for several months.
    • All parts need for low-cost device and gateway lists all the parts you need to build both the low-cost device and the gateway.
    • A leaflet for the IoT device is a leaflet summarizing the IoT device side.

    The next step is to program the LoRa part. This repository will give you the corresponding sketches.
